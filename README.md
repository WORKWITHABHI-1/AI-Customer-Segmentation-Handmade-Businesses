# 🤖 Artificial Intelligence-Driven Customer Segmentation and Personalized Marketing for Handmade Small Businesses

## 📖 MBA Project

This project demonstrates how Artificial Intelligence can be used to segment customers into meaningful groups using Machine Learning (K-Means Clustering). The objective is to help handmade small businesses understand customer purchasing behavior and improve marketing strategies through personalized recommendations.

---

## 🎯 Objectives

- Perform AI-based customer segmentation
- Analyze customer purchasing patterns
- Generate personalized marketing recommendations
- Visualize business insights through an interactive dashboard

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Plotly
- Streamlit

---

## 🤖 Machine Learning

**Algorithm Used:** K-Means Clustering

The AI model classifies customers into three business segments:

- 🟢 High Value Customers
- 🟡 Moderate Customers
- 🔴 Risk Customers
